# Hello-projekti

## Kurssin tiedot
- **Kurssi:** Git-versionhallinta
- **Toteutuskoodi:** SOF013AS2A-3001
- **Tekijä:** Roosa Kivilä

## Kuvaus

Tämä repositorio sisältää Git-versionhallinta -kurssin harjoitustehtävät. Projekti on yksinkertainen Hello World -verkkosovellus, jota on kehitetty kurssin aikana opetellen Gitin perustoimintoja.

## Sisältö

- **index.html** - Pääsivu, joka näyttää tervehdyksen ja kellonajan
- **styles.css** - Sivuston tyylimäärittelyt
- **scripts.js** - JavaScript-toiminnallisuus (analoginen kello)

## Opitut Git-komennot

Kurssin aikana harjoiteltiin mm. seuraavia Git-komentoja:

| Komento | Kuvaus |
|---------|--------|
| `git init` | Repositorion alustus |
| `git add` | Muutosten lisääminen staging-alueelle |
| `git commit` | Muutosten tallentaminen |
| `git status` | Tilan tarkistaminen |
| `git log` | Historian tarkastelu |
| `git branch` | Haarojen hallinta |
| `git checkout` / `git switch` | Haarojen vaihto |
| `git merge` | Haarojen yhdistäminen |
| `git revert` | Commitin peruuttaminen |
| `git restore` | Muutosten peruuttaminen |
| `git remote` | Etärepositorioiden hallinta |
| `git push` | Muutosten vienti etärepositorioon |
| `git fetch` | Muutosten haku etärepositoriosta |
| `git pull` | Muutosten haku ja yhdistäminen |
